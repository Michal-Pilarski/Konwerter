# Konwerter
Konwerter systemów liczbowych od binarnego do dziesiątkowego
W folderze dist znajduje się gotowy program uruchamiany po kliknięciu w ikonę
Systemy liczbowe do wyboru:
- binarny
- trójkowy
- czwórkowy
- piątkowy
- szóstkowy
- siódemkowy
- ósemkowy
- dziewiątkowy
- dziesiętny
- jedenastkowy
- dwunastkowy
- trzynastkowy
- szesnastkowy
- dwudziestkowy
